# CommerceFix CSV Repair

CommerceFix is a Shopify CSV Repair File Generator for store operators, VAs, migration helpers, and small agencies that need import-safe product CSV files before upload.

Live tool: https://soup-rolls.github.io/commercefix-csv-repair/

## What CommerceFix Does

- Scans a Shopify product CSV for import risks, SEO metadata gaps, image CSV issues, and variant field problems.
- Shows a limited free preview: issue counts, categories, and five sample rows.
- Generates paid repair-file packages after checkout.
- Keeps the workflow file-in / file-out.

## Paid Repair Files

CommerceFix paid exports are designed around inspectable files:

- `fixed_import_safe.csv`
- `seo_patch_only.csv`
- `error_report.xlsx`
- `before_after_preview.html`

Sample package page:

https://soup-rolls.github.io/commercefix-csv-repair/sample-report.html

## What CommerceFix Does Not Do

- No Shopify login.
- No Shopify API connection.
- No automatic import into Shopify.
- No SEO ranking guarantee.
- No account access, passwords, cookies, or private store credentials.

## Product Boundary

CommerceFix is an independent Shopify CSV repair-file generator. It is not Shopify, not an official Shopify app, and not a migration agency. The tool helps users inspect and repair CSV files before upload, but users should always review generated files before importing them into Shopify.
